# Overview
Bash commands written with Go

## File commands

### Basic
Lists files and directories
```
$ ls

$ ls path/filename/directory
```
More content
```
$ ls -la 
```
Cat
```
$ cat filename
```

### Word Count
Returns the number of lines in a file
```
$ wc -l
```
Returns the number of words in a file
```
$ wc -w
```
Returns the number of bytes in a file
```
$ wc -c
```
