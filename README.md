# Overview
Bash commands written with Go

## How to use
Clone the repository
```
$ git clone <https://github.com/bettaburger/grp>
```
Cd to the path
```
$ cd path/to/grp/cmd
```
Run the program
```
$ go run main.go
```

## Environment

```$ env ```
```$ setenv ```
```$ getenv ```
```$ unsetenv ```

## Basic
Lists files and directories
```
$ ls
```
