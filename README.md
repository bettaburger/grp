# Overview
A shell program with a few commands written in Go

## How to use
Clone the repository
```
$ git clone <https://github.com/bettaburger/grp>
```
Cd to the path
```
$ cd path/to/grp/cmd
```
Run the program
```
$ go run main.go
```
With Taskfile
```
$ task
```
